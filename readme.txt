A simple plugin for theme developers to maintain sanity when dealing with HTML email templates.
Convert any page generated by WP into HTML with inline styles to be used in newsletters.

== Installation ==

= Manual Installation =

1. Upload the entire `/sane-html-newsletters` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
